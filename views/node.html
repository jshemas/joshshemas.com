Basic Overview

Node.js is a platform built on Chrome's JavaScript Runtime(V8) for easily building fast, scalable network applications. Basically it's Javascript on the backend. More info can be found on their site.

Nodemon

When building a Node.js program, nodemon will watch the files in the directory in which nodemon was started, and if any files change, nodemon will automatically restart your node application.
