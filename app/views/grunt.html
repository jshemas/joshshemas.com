Basic Overview

Grunt is a Javascript task runner, that helps automate your web app. Instate of manual linting your files or minfing them, you can use Grunt to do this for you ever time you build. More info can be found on their site.

To Use:

First install Grunt-cli

And now you'll want to make a Grunt config file called 'Gruntfile.js' in your app directory. Example below.

Then to run your grunt file, just type grunt.
