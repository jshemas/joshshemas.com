Basic Overview

AngularJS is an open-source web application framework, maintained by Google and community, that assists with creating single-page applications, one-page web applications that only require HTML, CSS, and JavaScript on the client side. More info can be found on their site.

To Use:

You can download and include AngularJS in your source code, or you can use a CDN. Download Link.
